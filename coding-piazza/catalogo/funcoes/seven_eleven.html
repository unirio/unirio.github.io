Em um jogo de 7/11 (seven-eleven), o jogador lança um par de dados e soma seus resultados. Se a soma dos valores dos dois dados for 7 ou onze, o jogado ganha. Caso contrário, ele perde. Quantas vezes você precisa lançar os dados, em média, para ganhar no jogo? Escreva uma função que calcule este número a partir de uma simulação.
