Implementar uma janela que contenha três linhas de edição, sendo uma para entrada de uma temperatura em graus Celsius, outra para graus Fahrenheit e a terceira para graus Kelvin. Sempre que uma temperatura é digitada em uma linha de edição, a temperatura equivalente nas demais escalas deve ser apresentada nas outras linhas de edição.

Apenas para lembrar uma temperatura em graus Celsius pode ser convertida para graus Kelvin somando-se 273 graus (por exemplo, zero graus Celsius é equivalente a 273 graus Kelvin). Para converter de Celsius para Fahrenheit, multiplique a temperatura em graus Celsius por 1,8 e some 32.

Você pode implementar diversas variantes deste conversor, como:

  • Conversor de unidades de distância (m, km, jardas, ...)
  • Conversor de unidades de tempo (s, min, hr, ...)
  • Conversor de unidades de peso (kg, libras, ...)
  • Conversor de unidades de armazenamento (bytes, Kb, Mb, ...)
  • Conversor de moedas (dólar, reais, euros, ...)
