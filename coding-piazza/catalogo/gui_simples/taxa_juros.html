Implementar uma janela que contenha quatro linhas de edição, sendo as três primeiras editáveis e a última read-only. A primeira linha de edição será usada para que o usuário entre com um valor em dinheiro, a segunda para que ele entre com uma taxa de juros anual e a terceira para que ele entre com o número de anos para o empréstimo.

A janela deve ter ainda um botão que, quando pressionado, calcula o valor corrigido do empréstimo, que deve ser apresentado na linha de edição read-only. Vamos usar uma taxa de juros composta, ou seja, o valor ajustado do empréstimo deve ser igual ao seu valor original multiplicado por (1 + taxa) ^ tempo.
