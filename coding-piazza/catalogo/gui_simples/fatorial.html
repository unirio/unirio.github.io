Implementar uma janela que contenha duas linhas de edição, sendo a primeira editável e a segunda read-only. Toda vez que o usuário digitar um número na primeira linha de edição e pressionar ENTER, o fatorial do número digitado deve ser apresentado na linha de edição read-only. Se o valor digitado pelo usuário não for um número inteiro ou for maior do que 50, a linha de edição onde o valor foi digitado deve ser pintada de vermelho, retornando para sua cor padrão quando o usuário digitar qualquer coisa.

Algumas plataformas oferecem um componente com dois pequenos botões para incrementar ou decrementar um número. Se a sua plataforma oferecer este componente, adiciona na interface gráfica que você está construindo.
