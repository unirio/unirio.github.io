Implementar uma janela que contenha duas linhas de edição, sendo a primeira read-only e a segunda editável, dois botões e uma label.

Quando o primeiro botão é pressionado, o sistema sorteia um número entre 0 e 99 e apresenta dois asteriscos na linha de edição read-only. O usuário deve tentar adivinhar este número!

Para isto, o usuário digita um número na linha de edição editável e pressiona o segundo botão. Neste momento, o sistema deve incrementar um número de tentativas do usuário, aguardado internamente.

Se o número digitado for igual ao número sorteado pelo sistema, este número deve ser apresentado na linha de edição read-only e a label deve informar que o usuário acertou em X tentativas.

Se o número digitado for menor que o número sorteado pelo sistema, o sistema deve mostrar a mensagem "Tente um número menor" na label e o jogo continua. De forma similar, se o número digitado for maior que o número sorteado, o sistema deve mostra a mensagem "Tente um número maior".
