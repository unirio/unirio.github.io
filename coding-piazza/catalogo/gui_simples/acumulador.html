Implementar uma janela que contenha duas linhas de edição, sendo a primeira editável e a segunda read-only com o valor inicial "0". Toda vez que o usuário digitar um número na primeira linha de edição e pressionar ENTER, o valor deve ser somado ao valor da linha de edição read-only. Se o valor digitado pelo usuário não for um número, a linha de edição onde o valor foi digitado deve ser pintada de vermelho, retornando para sua cor padrão quando o usuário digitar qualquer coisa.
