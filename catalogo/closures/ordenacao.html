Ordenação: escreva um programa que ordene um vetor de números inteiros pelo número de divisores de cada número. Entre dois números com a mesma quantidade de divisores, o menor número deve aparecer primeiro na ordenação. 

[1, 2, 3, 4, 5, 6] -> [1, 2, 3, 5, 4, 6]
// 1 tem um divisor
// 2, 3 e 5 têm 2 divisores
// 4 tem três divisores
// 6 tem quatro divisores
