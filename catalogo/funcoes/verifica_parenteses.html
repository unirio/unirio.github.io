Escreva a função verificaParênteses que recebe uma string contendo sinais de abre e fecha parênteses, além de quaisquer outros caracteres. A função deve retornar TRUE se as aberturas de parênteses correspondem aos fechamentos (conforme os exemplos abaixo), ignorando os outros caracteres.

verificaParênteses("()") // true
verificaParênteses("((") // false
verificaParênteses("(())") // true
verificaParênteses("()()") // true
verificaParênteses("(()))") // false
verificaParênteses(")(") // false
